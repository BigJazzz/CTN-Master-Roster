How To

General

  1. Select the master roster start date you want to import
  2. Enter the line number to start from
  3. Adjust the dates as necessary
    • If you don't select the correct fortnight start, you'll be given a choice of the Sundays prior and after the date
    • The end date will always be be at least 14 days from the start date, and will default to the end of the fortnight for the date selected (if applicable)

General Instructions

  1. 1. Select the master roster start date you want to import.
  2. 2. Enter the line number to start from.
  3. 3. Adjust the dates as necessary.
    • If you don't select the correct fortnight start, you'll be given a choice of the Sundays prior and after the date.
    • The end date will always be at least 14 days from the start date, and will default to the end of the fortnight for the date selected (if applicable).
  4. 4. The app requires you to perform a test run first to ensure the data being pulled is accurate.
  5. 5. Once the test run is complete, the "Generate ICS Files" button will become available.
  6. 6. At least two files will be generated: one with your shifts and one with your days off.
    • If you request a significant number of lines, you may end up with more than one Shift file.
